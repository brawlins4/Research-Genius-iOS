# Research Genius iOS
<p>Research Genius is a tool designed by the library at Georgetown College using LibGuides CMS aimed at giving students an understanding of what it means to do college level research and can be customized to fit your institution's needs. It consists of three modules (Intro to Academic Research, Research as Inquiry, and Ethical Research). This version of Research Genius is a WebView application for iOS.  It is simple to customize, and can be used to create an app for your library or organization with a few changes and a URL.</p?
